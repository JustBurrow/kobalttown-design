Global Navigation Bar 공용 템플릿

공용 템플릿을 사용해 간단하게 네비게이션 바를 추가할 수 있습니다. 각 페이지에서 현재 페이지가 어떤 네비게이션 바를 표시할지는 공용 템플릿에서 판단합니다.

      <div th:fragment="gnb">
        <nav th:replace="_components/gnb/__${#authorization.expr('hasAuthority(''ROLE_ANONYMOUS'')') ? 'guest' : 'user'}__ :: navbar"></nav>
      </div>
